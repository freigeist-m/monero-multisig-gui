About

Monero Multisig GUI is a desktop app that creates and manages Monero multisignature wallets.

Full privacy and anonimity is preserved thanks to Tor network.

-- Each user has one or multiple Tor onions addresses used to coordinate the new multisigs.

-- To generate a new multisig, each user only needs to know other users's onion addresses and agree on a common referece.

-- Each user can propose new transfers over Tor to be signed by peers.

This tool makes multisig accessible to users who prefer an app over command-line tools.
